# Oasis_Spire_3D
A 3D Atmospheric Water Generator Simulation Built with C++ and OpenGL (GLUT).

## 🌟 项目简介 Overview
本项目是一个基于 C++ 和 GLUT/OpenGL 开发的 3D 图形学作品——**Oasis Spire（绿洲之塔）**。该项目不仅实现了复杂的 3D 几何体建模，还展示了图形学中的核心渲染技术，如光照模型、纹理贴图、空间阴影以及环境氛围的营造。

## 🛠️ 技术特点 Technical Features
- **3D 建模与层次建模 (Hierarchical Modeling)**：使用 OpenGL 基础图元与数学变换，构建了结构复杂的生物仿生几何建筑。
- **光照与材质 Lighting & Materials**：实现了经典的光照模型（如 Phong/Blinn-Phong），包含环境光、散射光和镜面反射，表现出建筑物不同材质的质感。
- **纹理映射 Texture Mapping**：对塔身、地面以及环境特效应用了自定义纹理贴图，增强了画面的真实感与细节。
- **交互控制 Interactions**：支持键盘/鼠标交互，用户可以自由旋转、缩放视角，全方位观察 3D 模型。

## 🚀 运行环境与依赖 (Requirements)
- **语言**：C++ (C++11 或更高)
- **图形库**：OpenGL, GLUT / FreeGLUT
- **IDE 推荐**：Visual Studio / CLion

## 🏃‍♂️ 快速运行指南 (Usage)
1. 确保你的开发环境已正确配置好 **GLUT/FreeGLUT** 库。
2. 克隆本仓库到本地。
3. 将 `assets/` 文件夹保持在与编译出的可执行文件相同的目录下（确保程序能正确加载纹理图片）。
